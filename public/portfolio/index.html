Dashboards

Mail server load dashboard

  • Deployed

This dashboard was build to control the burden on an email server doing marketing campaign, and prevent it from overloading. The capacity was around 500 emails per hour, and campaigns were sent all day long. So it was necessary to constantly get the data from the mail server. It is built in Python Plotly Dash, and was deployed on a private server, this is a sample app, using sample data, deployed on heroku. It also has better funnier colorscheme (: Source code is available on Github.
